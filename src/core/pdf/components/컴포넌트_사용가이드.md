# components.py 가이드

뉴스레터 PDF를 구성하는 콘텐츠 단위 객체 모음.
모든 클래스는 ReportLab의 `Flowable`을 상속하며, `NewsletterDocument.build(story)`에 리스트로 전달된다.

---

## 목차

- [Divider](#divider)
- [ArticleBlock](#articleblock)
- [ImageBlock](#imageblock)
- [NoticeBlock](#noticeblock)

---

## Divider

기사와 기사 사이를 구분하는 수평선.

```python
Divider()
```

파라미터 없음. 색상(`#cccccc`), 두께(`1pt`), 상하 여백(`4mm`)이 고정 적용된다.

**사용 예시**

```python
story = [
    ArticleBlock(...),
    Divider(),
    ArticleBlock(...),
]
```

## ArticleBlock

제목과 본문으로 구성된 기사 블록.

```python
ArticleBlock(
    title: str,
    body: str,
    styles: NewsletterStyleSheet,
)
```

| 파라미터 | 타입 | 설명 |
|---|---|---|
| `title` | `str` | 기사 제목 |
| `body` | `str` | 기사 본문. HTML 태그(`<b>`, `<i>` 등) 사용 가능 |
| `styles` | `NewsletterStyleSheet` | 폰트/색상 스타일 객체 |

**사용 예시**

```python
styles = NewsletterStyleSheet()

ArticleBlock(
    title="1학기 개강 안내",
    body="2026년 3월 2일부터 1학기 수업이 시작됩니다.",
    styles=styles,
)
```

## ImageBlock

이미지와 선택적 캡션으로 구성된 블록. 이미지는 가운데 정렬된다.

```python
ImageBlock(
    path: str,
    caption: str = "",
    max_width: float = 400,
)
```

| 파라미터 | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `path` | `str` | — | 이미지 파일 경로 |
| `caption` | `str` | `""` | 이미지 하단 캡션. 빈 문자열이면 캡션 미표시 |
| `max_width` | `float` | `400` | 이미지 최대 너비 (pt 단위) |

**사용 예시**

```python
ImageBlock(
    path="assets/event.png",
    caption="체육대회 행사 현장",
)

# 캡션 없이
ImageBlock(path="assets/logo.png")
```

## NoticeBlock

배경색이 있는 공지/강조 박스. 둥근 모서리(`radius=4`)와 하늘색 배경(`#e8f4fd`)이 적용된다.

```python
NoticeBlock(text: str)
```

| 파라미터 | 타입 | 설명 |
|---|---|---|
| `text` | `str` | 공지 내용. HTML 태그 사용 가능 |

**사용 예시**

```python
NoticeBlock("2026년 3월호 BSSM 뉴스레터에 오신 것을 환영합니다.")
```

## Story 구성 예시

```python
story = [
    NoticeBlock("환영합니다."),
    Divider(),
    ArticleBlock(title="1학기 안내", body="...", styles=styles),
    ImageBlock(path="assets/photo.png", caption="행사 현장"),
    Divider(),
    ArticleBlock(title="동아리 모집", body="...", styles=styles),
]

doc.build(story)
```
